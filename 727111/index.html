Power Analyser CASSY Plus

727111

The Power Analyser CASSY is a combination of isolated and differential oscilloscope, multimeter, wattmeter, energy analyzer and recorder. It was designed in its concept for demonstration and laboratory experiments.

The Power Analyser CASSY differs from its previous model in that it has an additional LAN connection

The Power Analyser CASSY has been optimized for the following applications:

Power grids:

  • Voltage and frequency stability
  • Load behavior of networks
  • Effect of harmonics

Electrical machines:

  • Inrush current of transformers and machines
  • Ratio of transformers
  • Efficiency of machines

Power Electronics:

  • rectifier
  • DC / DC converter
  • DC / AC converter
  • frequency converter
  • filter

POWER ANALYSER CASSY - IN DETAIL

  • simultaneous measurement of U, I, φU, φI, f and P
    • instantaneous values U, I and P
    • averaged values U, I and P
    • RMS values (AC+DC) U and I
    • fundamental wave filters
    • Delta connection adjustment
    • Measuring accuracy U,I is at 0,5%.
    • Frequency response voltage: 100 kHz 3 dB 250 V
    • Frequency response current: 40 kHz 3 dB at 10 A  
  • universal connection options
  • automatic or manual range selection
  • Support of the successful measurement software CASSY Lab 2 for Drives and Power Systems 524    222 for computer-aided measurements and simple to highly complex evaluations
    • Electrical power calculation S, P, QC and QL
    • Electrical work WS, W and WQ
    • Resistance calculation R, Z, XC, XL, G, Y BC and BL
    • Positive sequence component, negative sequence component and zero sequence component in 3-phase systems
    • Time derivative, integral over time, FFT analysis, mean value, histogram, and modelling
    • Drivers for LabVIEW ™ and MATLAB® available
  • Possibility of direct manual operation of the device by means of a rotary selector with cursor keys direct value readings on 9 cm backlit display
    • Display of up to 24 measured values on one display
    • Display of all values for each channel
    • Display of all values in tabular form
    • Display of measured values in a diagram
    • Display of a vector diagram
  • Measuring instrument category CATIII 300: allows the use of the measuring instrument for
    tests with safety extra-low voltage (SELV) via 3-phase systems with or without neutral conductor, up to testing in power electronics, e.g. DC link voltage of 700 V DC
  • Real-time processing in the device enables comprehensive network analysis in the three-phase networks, which are displayed directly on the device in the vector diagram

Digital and media connection

The Power Analyser CASSY is fully controllable in real time via the following interfaces:

  • RJ45 Ethernet (IP4 / IP6 networks)
  • WiFi IEEE 802.11 a/b/g/n supports WPA, WPA2, WPA3 and Radius Server
  • USB-C for direct PC connection or for external data carriers, also for simplified configuration.

These interfaces can be used by the following software, among others.

  • CASSY Lab 2 for drives and energy systems,
  • MATLAB® and LabVIEW ™ with freely available examples,
  • Lab Docs Editor Advanced / Pro for generating individual experiment instructions in HTML and with JavaScript even without programming experience.


The following integrated servers are available in the Power Analyser CASSY for the local media connection of up to 3 end devices simultaneously.

  • TCP server for the CASSY app
  • DHCP server for use with PC direct connection or as Wifi hotspot
  • HTTP server for the CASSY Web App
  • WebSocket server for WebSocket clients, such as Leybold Lab Doc
  • VNC server for 1:1 transmission of screen contents to remote devices, including a return channel for control purposes

PLUS:

Isolation amplifiers:

  • 4-channel isolation amplifier with the possibility of outputting I, U, UxI to the assigned output U, V, W and X.
  • Mathematical functions between the channels (A + B, AxB, etc)

Function generator:

  • 1 output channel Y as function generator
  • DC / sine / rectangle / triangle and free function

DISPLAY & OPERATION

  • Graphic display: 9 cm (3.5 "), QVGA, colored, light (adjustable up to 400 cd / m²)
  • Operation: Pushbutton and incremental encoder with pushbutton

INPUTS & OUTPUTS

  • Inputs: 4 isolated measuring channels CAT III 300, each with I and U
  • measurement (max. 8 can be used at the same time)
  • Input A -D: U and I connection via 4 mm safety sockets
  • Measuring ranges U: 25/70/250/700 VAC ± 36 / ± 100 / ± 360 / ± 1000 VDC
  • Measuring ranges I: 0.7 / 1.6 / 7/16 / AAC ± 1 / ± 2.5 / ± 10 / ± 16 ADC
  • Sampling rate: max. 1,000,000 values / s per channel at U and I max. 500,000 values / s
  • Analog outputs: A-D ± 10 V 200 mA

FUNCTION GENERATOR

  • Frequency range: 10 mHz ... 20 kHz
  • Amplitude: ± 10 V 200 mA
  • Waveform: Sine, Rectangle, Pulse Width, DC Offset, Triangle and Free Function fX
  • Resolution of the analog inputs: 16 bits
