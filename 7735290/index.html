Converter Controller CASSY

7735290

Multifunctional control unit for drive technology.

Scalar frequency converter

Control unit with microcontroller for the assembly of a frequency converter in accordance with the PWM characteristic curve method in operation together with the universal converter. A three-phase pulse width modulator controls the six power transistors of the universal converter and thus generates a sinusoidally-shaped motor current. The r.m.s. value of the motor voltage is set in accordance with a programmable, dynamically adapting U/f characteristic. Can be parametrised via 14 menu items. Display of current, voltage or frequency desired/actual value as well as all parameters via four-digit LED display. Desired value input either via external analogue interface -10 V...+10 V or internal manual input via rotary encoder.

Block commutation

Control unit with microcontroller for the assembly of a highly-dynamic AC servo drive with block signal-shape commutation in operation together with the permanently excited brushless DC machine (AC servo), the universal converter and the commutation pick-up. Alternatively, a synchronous or a multifunction machine 0.3 kW can be used. The pulse width modulation is selected so that block-shaped currents are generated in the motor windings. The digital controllers for current, speed and positioning are arranged in cascade configuration. The actual speed value is generated internally out of the commutation signals or measured with an incremental tacho-generator. Commutation signals can also be used for positioning with low resolution (30 degrees). Higher resolution servo positioning is possible by connecting an external servo controller via the parallel interface and an external position pick-up.

Sine commutation

Control unit with microcontroller for the assembly of highly-dynamic AC servo drives with sinusoidal signal-shape commutation in operation together with the permanently excited brushless DC machine (AC servo), the universal converter 735297 and the resolver 731094. Alternatively, a synchronous or a multifunction machine 0.3 kW can be used. The pulse width modulation is designed so that sinusoidal shaped currents are generated in the motor windings. The digital controllers for current, speed and positioning are arranged in cascade configuration. The actual speed value is generated internally out of the resolver signals. The resolver permits positioning with high resolution (10 bit/360 degrees, corresponding to 0.35 degrees).

Position Controller

The microprocessor-controlled digital position controller, in conjunction with the AC servo drive with block commutation, is used to set up a high-precision single-axis positioning drive. Actual positions can be recorded with the incremental tacho 0.3 (773 1092), which supplies 1024 pulses per rotation.

Measuring system:

The measuring system corresponds to that of the 727100 Power Analyser CASSY, a combination of a potential-free and differential oscilloscope, multimeter, wattmeter, energy analyser and recorder. Its concept has been designed for demonstration and laboratory testing.

The measuring system has been optimised for the application area of drive technology:
